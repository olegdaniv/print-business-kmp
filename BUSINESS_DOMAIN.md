# PrintBusinessKmp — Бізнес-домен

## 1. Огляд бізнесу

**Тип:** ФОП, група 3 спрощеної системи оподаткування (5% від доходу + ЄСВ ~1 900 грн/міс)

**Спеціалізація:** Друк на сувенірній та рекламній продукції з використанням двох ключових технологій — DTF (Direct-to-Film) та UV DTF (UV Direct-to-Film).

**Локація:** Львів, Україна

**Модель виробництва:** На поточному етапі — аутсорсинг друку у партнерів. Бізнес виступає як сервіс із прийому замовлень, підготовки дизайнів, закупки трансферів/стікерів у партнерів, перенесення на вироби (DTF) та продажу готової продукції.

**Канали продажу:** Власний вебсайт/інтернет-магазин (у розробці)

## 2. Технології друку

### 2.1 DTF (Direct-to-Film) — друк на текстиль

**Що це:** Технологія термічного перенесення зображень на тканину через спеціальну PET-плівку. Дизайн друкується на плівку пігментними чорнилами, посипається термоклеєм (порошком), полімеризується, а потім переноситься на виріб за допомогою термопресу.

**Технічні параметри перенесення:**

| Параметр | Значення |
|----------|----------|
| Температура термопресу | 160–170°C (залежно від плівки та порошку) |
| Час притиску | 15–20 секунд |
| Тиск | Середній–високий |
| Холодне відшарування | Після повного охолодження зняти плівку |
| Гаряче відшарування | Зняти плівку відразу (залежно від типу плівки) |
| Повторний притиск | 5–10 сек через тефлонову прокладку або пергамент |

**Сумісні матеріали:** Бавовна, поліестер, суміші, джинс, шкіра, фетр, фліс, нейлон — практично будь-який текстиль.

**Характеристики якості:**
- Витримує 50+ прань без втрати яскравості
- Повнокольоровий друк (CMYK + білий)
- Висока деталізація, фотореалістичність
- Тонкий тактильний шар на виробі
- Стійкість до УФ-випромінювання

**Основні продукти DTF:**
- Футболки (бавовна, поліестер, суміші)
- Худі та світшоти
- Шопери та еко-сумки
- Кепки та бейсболки
- Фартухи
- Рюкзаки
- Уніформа та спецодяг
- Рукавички, шарфи та інші текстильні аксесуари

### 2.2 UV DTF — друк на тверді поверхні

**Що це:** Технологія створення готових стікерів/наклейок за допомогою UV-чорнил, які друкуються на спеціальну плівку шарами (CMYK + білий + лак), а потім ламінуються. Результат — готовий самоклеючий стікер, який переноситься без термопресу (peel-and-stick).

**Процес нанесення:**
1. Очистити поверхню (мильна вода або 50/50 спирт+вода)
2. Зняти захисну підкладку зі стікера
3. Розмістити на поверхні
4. Притиснути шпателем або карткою, видавивши бульбашки повітря
5. Зняти верхню транспортну плівку
6. Дати закріпитися 24 години для максимальної адгезії

**Характеристики:**
- Термін служби 2–5 років на непористих поверхнях
- Водостійкість, стійкість до подряпин
- 3D-ефект (об'ємність завдяки шарам лаку)
- Глянцевий фініш
- Повнокольоровий друк (CMYK)
- Не потребує термопресу

**Сумісні матеріали:** Скло, метал, дерево, пластик, кераміка, акрил, камінь, шкіра (тверда), PVC.

**Основні продукти UV DTF:**
- Брендовані чашки, кухлі, термоси
- Пляшки (скло, метал)
- Телефонні кейси
- Брелоки
- Ручки, USB-флешки
- Блокноти та щоденники (тверді обкладинки)
- Коробки та пакування
- Вивіски та таблички
- Декоративні вироби (підставки, рамки)
- Корпоративні подарунки та промо-продукція

### 2.3 Порівняння DTF vs UV DTF

| Характеристика | DTF | UV DTF |
|----------------|-----|--------|
| Призначення | Текстиль (м'які поверхні) | Тверді поверхні |
| Метод нанесення | Термопрес (160–170°C) | Peel-and-stick (без нагріву) |
| Обладнання для нанесення | Термопрес обов'язковий | Не потрібно (шпатель) |
| Чорнила | Пігментні водні DTF-чорнила | UV-чорнила (ультрафіолетове затвердіння) |
| Тактильні відчуття | Тонкий шар на тканині | Об'ємний 3D-стікер |
| Стійкість | 50+ прань | 2–5 років на поверхні |
| Мінімальний тираж | Від 1 шт | Від 1 шт |

## 3. Послуги

### 3.1 Друк на одязі клієнта (DTF)

**Опис:** Клієнт приносить або надсилає власний виріб (футболку, худі тощо), ми наносимо його дизайн або розробляємо дизайн під замовлення.

**Робочий процес (workflow):**
1. **Прийом замовлення** — клієнт надсилає/приносить виріб + макет або опис ідеї
2. **Узгодження дизайну** — підготовка макету, затвердження з клієнтом
3. **Замовлення трансферу** — розміщення замовлення у партнера-друкарні на DTF-плівку
4. **Отримання трансферу** — перевірка якості плівки
5. **Перенесення** — термопресування на виріб клієнта
6. **Контроль якості** — перевірка адгезії, кольорів, відсутності дефектів
7. **Видача/доставка** — повернення готового виробу клієнту

**Цінові компоненти:**
- Вартість DTF-трансферу у партнера (за погонний метр або за площу)
- Вартість роботи з перенесення (термопрес)
- Вартість підготовки дизайну (якщо потрібно)
- Доставка (якщо є)

**Ризики:** Зіпсований виріб клієнта → потрібна компенсація або заміна.

### 3.2 Продаж готового одягу з принтом (DTF)

**Опис:** Закупка заготовок (blank-одягу), нанесення принту, продаж як готового товару.

**Робочий процес:**
1. **Закупка заготовок** — футболки, худі тощо у постачальників (Stedman, Fruit of the Loom, інші)
2. **Підготовка дизайну** — створення або адаптація дизайну
3. **Замовлення DTF-трансферу** — розміщення замовлення у партнера
4. **Перенесення** — термопресування на заготовку
5. **Складування/каталогізація** — готовий товар на склад
6. **Продаж** — через сайт, соцмережі, або офлайн
7. **Доставка** — відправка замовлення клієнту (Нова Пошта, Укрпошта)

**Цінові компоненти:**
- Закупна вартість заготовки (garment cost)
- Вартість DTF-трансферу у партнера
- Вартість роботи з перенесення
- Дизайн (амортизований на тираж)
- Упаковка та доставка
- Маржа

**Моделі продажу:**
- **Під замовлення (print-on-demand)** — клієнт обирає дизайн + розмір → друк після оплати
- **Готовий склад** — популярні дизайни заздалегідь надруковані в кількох розмірах
- **Обмежені серії (limited drops)** — обмежений тираж для створення попиту

### 3.3 UV DTF стікери на тверді поверхні

**Опис:** Виготовлення та продаж UV DTF стікерів/наклейок для брендування твердих поверхонь.

**Робочий процес:**
1. **Прийом замовлення** — клієнт надає макет + вказує розмір і кількість
2. **Підготовка макету** — адаптація дизайну під UV DTF (CMYK, розкладка на ганг-шиті)
3. **Замовлення у партнера** — розміщення замовлення на UV DTF друк
4. **Отримання стікерів** — перевірка якості
5. **Нарізка** — якщо потрібно, розрізання ганг-шиту на окремі стікери
6. **Пакування та видача/доставка**

**Або:** Клієнт замовляє стікери для самостійного нанесення (без виробу).

**Цінові компоненти:**
- Вартість UV DTF друку у партнера (за площу / за ганг-шит)
- Вартість підготовки дизайну
- Нарізка (якщо потрібно)
- Упаковка та доставка

## 5. Партнери та аутсорсинг

### 5.1 Роль партнерів у бізнес-моделі

На поточному етапі партнери-друкарні є критичною ланкою бізнесу. Вони виконують друк DTF-трансферів та UV DTF стікерів. Бізнес залежить від їхніх цін, якості та термінів.

### 5.2 Типи партнерів

| Тип партнера | Що надає | Приклади у Львові/Україні |
|--------------|----------|--------------------------|
| DTF-друкарня | DTF-трансфери (плівка з принтом) | PNG druk, MARK.HUB, Poster DTF, BrandME |
| UV DTF друкарня | UV DTF стікери | PNG druk, BrandME |
| Постачальник заготовок | Blank-одяг (футболки, худі) | Stedman, Fruit of the Loom, локальні оптовики |
| Постачальник сувенірки | Чашки, брелоки, ручки тощо | Оптові постачальники, Alibaba |
| Дизайнер (за потребою) | Графічний дизайн | Фрілансери, студії |

### 5.3 Дані для моделі Partner

Для кожного партнера потрібно відслідковувати:
- Контактна інформація (назва, телефон, email, адреса, менеджер)
- Типи послуг (DTF друк, UV DTF друк, заготовки)
- Прайс-лист (ціна за пог. метр DTF, ціна за ганг-шит UV DTF тощо)
- Мінімальне замовлення (якщо є)
- Середній термін виконання
- Якість (рейтинг, нотатки)
- Історія замовлень та витрат

## 6. Фінансова модель

### 6.1 Структура собівартості (DTF на одязі)

```
Собівартість одиниці = 
    Вартість заготовки (якщо ми надаємо одяг)
  + Вартість DTF-трансферу у партнера
  + Вартість роботи (перенесення, підготовка)
  + Частка накладних витрат (електрика, оренда, амортизація термопресу)
  + Упаковка
```

### 6.2 Структура собівартості (UV DTF стікери)

```
Собівартість одиниці = 
    Вартість UV DTF друку у партнера (за площу)
  + Вартість нарізки (якщо потрібно)
  + Вартість роботи
  + Упаковка
```

### 6.3 Ціноутворення

| Параметр | Опис |
|----------|------|
| costPrice | Повна собівартість одиниці |
| marginPercent | Бажана маржа (40–65%) |
| suggestedPrice | costPrice × (1 + marginPercent) |
| taxPercent | Податок ФОП 5% (від доходу, не від прибутку!) |
| finalPrice | Ціна для клієнта |
| profit | finalPrice − costPrice − (finalPrice × taxPercent) |

**Важливо:** ФОП група 3 платить 5% від УСЬОГО доходу, а не від прибутку. Це означає, що при розрахунку реального прибутку потрібно враховувати:
- 5% єдиний податок від повної суми продажу
- ЄСВ ~1 900 грн/міс (фіксований, не залежить від доходу)

### 6.4 Приклади розрахунку

**DTF футболка (з нашою заготовкою):**
- Заготовка (Stedman бавовна): ~150 грн
- DTF-трансфер (A4, у партнера): ~50 грн
- Робота + накладні: ~30 грн
- **Собівартість: ~230 грн**
- Маржа 50%: ціна ~345 грн
- Податок 5%: ~17 грн
- **Прибуток: ~98 грн з футболки**

**UV DTF стікер (набір 10 шт):**
- UV DTF друк у партнера: ~80 грн
- Нарізка + робота: ~20 грн
- **Собівартість: ~100 грн**
- Маржа 60%: ціна ~160 грн
- Податок 5%: ~8 грн
- **Прибуток: ~52 грн з набору**

*Ціни орієнтовні і залежать від конкретних партнерів та обсягів.*

## 7. Доменна модель — сутності системи

### 7.1 Поточні сутності (вже реалізовано)

- **Client** — клієнт (фізична або юридична особа)
- **Order** — замовлення з лайн-айтемами
- **OrderItem** — позиція замовлення з ціноутворенням
- **Invoice** — рахунок-фактура
- **Layout** — шаблон дизайну
- **Partner** — партнер-аутсорсер
- **OutsourceJob** — замовлення у партнера
- **BusinessProfile** — налаштування ФОП

### 7.2 Рекомендовані розширення

#### Product (Товар/Заготовка)
Окрема сутність для управління каталогом заготовок та готових товарів.

```
Product {
    id
    name               // "Футболка Stedman Classic бавовна біла"
    category            // GARMENT, SOUVENIR, ACCESSORY
    productType         // T_SHIRT, HOODIE, MUG, STICKER, etc.
    supplier            // Постачальник
    purchasePrice       // Закупна ціна
    sizes[]             // Доступні розміри (для одягу)
    colors[]            // Доступні кольори
    stockQuantity       // Поточний залишок
    minStockLevel       // Мінімальний залишок (для сповіщень)
    imageUrl
    notes
}
```

#### Design (Дизайн/Макет)
Розширення поточного Layout — додати зв'язок із замовленнями та версіонування.

```
Design {
    id
    name
    serviceType         // DTF, UV_DTF
    category            // CUSTOM (клієнтський), CATALOG (наш каталог), TEMPLATE
    dimensions          // ширина × висота (см)
    dpiResolution       // 300+ DPI
    fileUrl             // Посилання на файл макету
    previewUrl          // Превʼю для сайту
    status              // DRAFT → APPROVED → IN_PRODUCTION → ARCHIVED
    clientId?           // Якщо кастомний — для якого клієнта
    tags[]              // Теги для пошуку ("патріотичне", "мінімалізм", "кави")
    createdAt
}
```

#### OutsourceOrder (Замовлення у партнера) — розширення OutsourceJob
Більш детальне відстеження замовлень у партнерів-друкарень.

```
OutsourceOrder {
    id
    partnerId
    orderIds[]          // Зв'язок із клієнтськими замовленнями
    serviceType         // DTF_PRINT, UV_DTF_PRINT
    specifications      // Площа друку, кількість, тип плівки
    totalArea           // Загальна площа друку (м²)
    costFromPartner     // Вартість від партнера
    status              // PENDING → SENT → IN_PRODUCTION → READY → RECEIVED → QUALITY_CHECK
    orderedAt
    expectedDelivery
    receivedAt
    qualityNotes        // Нотатки з перевірки якості
}
```

#### Delivery (Доставка)
Для відстеження доставки замовлень клієнтам.

```
Delivery {
    id
    orderId
    method              // PICKUP, NOVA_POSHTA, UKRPOSHTA, COURIER
    trackingNumber
    cost
    status              // PENDING → SHIPPED → DELIVERED
    address
    recipientName
    recipientPhone
}
```

// Статус замовлення — розширений під повний workflow
enum OrderStatus {

## 8. Бізнес-правила

### 8.1 Замовлення

1. Замовлення може містити позиції різних типів (DTF + UV DTF в одному замовленні)
2. Для DTF із одягом клієнта — відслідковувати джерело заготовки (`GarmentSource`)
3. Мінімальна ціна замовлення — налаштовувана (наприклад, 100 грн)
4. Замовлення не може перейти у виробництво без затвердженого дизайну
5. Скасування можливе до етапу `OUTSOURCE_ORDERED` (після — з штрафом або неможливе)

### 8.2 Дизайн

1. Мінімальна роздільна здатність: 300 DPI (рекомендовано 400)
2. Колірна модель: CMYK для друку
3. Формат файлу: PNG (прозорий фон), PDF, AI, CDR
4. Для DTF — макет дзеркалити не потрібно (партнер робить це сам)
5. Для UV DTF — узгодити з партнером вимоги до розкладки (ганг-шит)

### 8.3 Ціноутворення

1. Податок 5% нараховується на повну суму продажу (дохід), не на прибуток
2. ЄСВ розподіляється по місяцях, не по замовленнях
3. Знижки: можливі для B2B-клієнтів та великих тиражів (налаштовуване)
4. Всі ціни у гривнях (UAH / грн)
5. Округлення до 2 знаків після коми

### 8.4 Якість

1. Кожен DTF-трансфер перевіряється перед нанесенням (колір, різкість, дефекти)
2. Після перенесення — тест на адгезію (потягнути край)
3. UV DTF стікери — перевірити рівномірність ламінації та адгезії
4. Бракований виріб → повторне виробництво за наш рахунок
